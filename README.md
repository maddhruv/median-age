# median-age

Countries by Median Age :older_man: :woman: :boy: :baby: - extracted from https://en.wikipedia.org/wiki/List_of_countries_by_median_age

## Installation

`npm i median-age`

## Usage

### Module

```js
import { data } from "median-age";

console.log(data);
```

### CLI

```shell
npx median-age
```
